## Contributing a Translation

Want to help translate NotionSync? Here's how:

1. **Check if your language exists** - Look in this folder
2. **If it exists** - Review and improve the AI-generated draft
3. **If it doesn't exist** - Create a new file `README.[lang-code].md`

### Translation Guidelines

- Maintain all markdown formatting
- Keep code blocks unchanged
- Translate technical terms appropriately for your locale
- Keep URLs and links as-is
- Preserve emoji and badges

### How to Submit

1. Fork the repository
2. Create or edit the translation file
3. Submit a Pull Request with title: `translation: add/improve [language] README`
4. Tag your PR with `documentation` and `translation` labels
